For any probability distribution on \(\sigma \in S_L\) and any fixed \(\hat\pi \),

\[ \mathrm{cost}(\hat\pi ) = \sum _{i{\lt}j} \Big( p_{ij}\, \mathbf{1}[\hat\pi (i) {\gt} \hat\pi (j)] + (1-p_{ij})\, \mathbf{1}[\hat\pi (i) {\lt} \hat\pi (j)] \Big), \]

where \(p_{ij} = \mathbb {P}(\sigma (i) {\lt} \sigma (j))\).

Proof ▶

For each pair \(i{\lt}j\) let \(X_{ij}(\sigma ,\hat\pi ) = \mathbf{1}[\sigma (i){\lt}\sigma (j) \text{ and } \hat\pi (i){\gt}\hat\pi (j)] + \mathbf{1}[\sigma (i){\gt}\sigma (j) \text{ and } \hat\pi (i){\lt}\hat\pi (j)]\). Then \(X_{ij} = 1\) exactly when \(\sigma \) and \(\hat\pi \) disagree on the relative order of \(i\) and \(j\), so \(d_{\mathrm{KT}}(\hat\pi ,\sigma ) = \sum _{i{\lt}j} X_{ij}\). Taking the posterior expectation and using that \(\hat\pi \) is fixed (so the two indicator events on \(\hat\pi \) are constants pulled out of the expectation, and \(\mathbb {P}(\sigma (i){\gt}\sigma (j)) = 1 - p_{ij}\) since \(\sigma \) is a permutation) gives the formula. The normalization by \(Z \ne 0\) is what turns the weighted sums into the probabilities \(p_{ij}\).

Lemma 8 Majority form of a pairwise contribution

  • LeaFrontier.OrderingRecovery.majority_form

Write the contribution of pair \(\{ i,j\} \) to \(\mathrm{cost}(\hat\pi )\) as \(\ell _{ij}(\hat\pi ) = p_{ij}\, \mathbf{1}[\hat\pi (i){\gt}\hat\pi (j)] + (1-p_{ij})\, \mathbf{1}[\hat\pi (i){\lt}\hat\pi (j)]\). Calling the more likely relative order the posterior majority order (ties broken arbitrarily),

\[ \ell _{ij}(\hat\pi ) = |2p_{ij}-1|\, \mathbf{1}[\hat\pi \text{ disagrees with the majority order on } \{ i,j\} ] + \min (p_{ij}, 1-p_{ij}). \]

In particular, if \(p_{ij} = 1/2\) the contribution is always \(1/2\), independent of \(\hat\pi \).

Proof ▶

If \(\hat\pi \) agrees with the posterior majority order, \(\ell _{ij}(\hat\pi ) = \min (p_{ij}, 1-p_{ij})\); if it disagrees, \(\ell _{ij}(\hat\pi ) = \max (p_{ij}, 1-p_{ij})\). Since \(\max (p,1-p) = \min (p,1-p) + |2p-1|\) for every \(p \in [0,1]\), the claimed expression follows. At \(p_{ij} = 1/2\) the coefficient \(|2p_{ij}-1|\) vanishes and the constant term is \(1/2\).
