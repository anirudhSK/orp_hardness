0.1 Layer 0 — Core definitions

Definition 1 Tournament and degree difference
✓
#

Lean declarations

  • LeaFrontier.OrderingRecovery.Tournament

A tournament on \(V = \{ 1,\dots ,n\} \) (\(= \mathrm{Fin}\, n\)) is a relation \(\mathrm{beats} : V \to V \to \mathrm{Bool}\) that is irreflexive (\(\mathrm{beats}(u,u) = \mathrm{false}\)) and exactly one-directional on each unordered pair (\(u \ne v \Rightarrow \mathrm{beats}(u,v) = \neg \, \mathrm{beats}(v,u)\)). Arc \((u,v) \in A\) means \(\mathrm{beats}(u,v) = \mathrm{true}\). For \(u \in V\) the degree difference is \(\delta _u := \operatorname {indeg}(u) - \operatorname {outdeg}(u)\) (Tournament.degDiff).

Lemma 2 Degree-difference bound
✓
#

Uses

  • Definition 1

Lean declarations

  • LeaFrontier.OrderingRecovery.degDiff_abs_le

For every vertex \(u\) of a tournament on \(n\) vertices, \(|\delta _u| \le n - 1\).

Proof ▶

Each of the \(n-1\) vertices \(v \ne u\) contributes exactly \(+1\) or \(-1\) to \(\operatorname {indeg}(u) - \operatorname {outdeg}(u)\), according to whether \((v,u) \in A\) or \((u,v) \in A\). Hence the signed sum has absolute value at most \(n-1\).

Definition 3 FAST objective and decision problem
✓
#

Uses

Lean declarations

  • LeaFrontier.OrderingRecovery.FAST

For a linear order \(\pi \in S_V\) (an Equiv.Perm of \(V\), with \(\pi (u)\) the position of \(u\)), the feedback-arc-set count is \(\operatorname {fas}(T,\pi ) = |\{ (u,v) \in A : \pi (u) {\gt} \pi (v)\} |\). The FAST decision predicate is \(\mathrm{FAST}(T,k) :\Leftrightarrow \exists \pi ,\ \operatorname {fas}(T,\pi ) \le k\).

For \(\hat\pi , \sigma \in S_L\), \(d_{\mathrm{KT}}(\hat\pi ,\sigma ) = \sum _{i{\lt}j} \mathbf{1}[(\sigma (i)-\sigma (j))(\hat\pi (i)-\hat\pi (j)) {\lt} 0]\), the number of pairs on which \(\hat\pi \) and \(\sigma \) disagree.

Definition 5 Posterior weight, normalizer, and pair marginal
✓
#

Lean declarations

  • LeaFrontier.OrderingRecovery.ORPInstance.pairLess

An ORP instance carries \(L\) finite-support rational noise PMFs \(F_i : \mathbb {Z} \to \mathbb {Q}\). With every observation equal to \(0\), the unnormalized posterior weight of \(\sigma \in S_L\) is \(w(\sigma ) = \prod _{i} F_i(-\sigma (i))\) (ORPInstance.weight), normalized by \(Z = \sum _\sigma w(\sigma )\) (ORPInstance.Z). For events \(i,j\) the pair marginal is \(p_{ij} = \mathbb {P}(\sigma (i) {\lt} \sigma (j)) = Z^{-1}\sum _{\sigma : \sigma (i) {\lt} \sigma (j)} w(\sigma )\) (ORPInstance.pairLess).

The posterior cost of a candidate order \(\hat\pi \) is \(\mathrm{cost}(\hat\pi ) = Z^{-1}\sum _\sigma w(\sigma )\, d_{\mathrm{KT}}(\hat\pi ,\sigma )\) (ORPInstance.cost). The ORP decision predicate is \(\mathrm{ORP}(\mathrm{inst},K) :\Leftrightarrow \exists \hat\pi , \mathrm{cost}(\hat\pi ) \le K\).
