Theorem 17Permutation fiber-count

For a finite type \(\alpha \), an embedding \(e : \mathrm{Fin}\, n \hookrightarrow \alpha \), and any \(F\),

\[ \sum _{\sigma : \mathrm{Perm}(\alpha )} F(\lambda i.\, \sigma (e\, i)) = (|\alpha |-n)!\, \cdot \sum _{\varphi : \mathrm{Fin}\, n \hookrightarrow \alpha } F(\lambda i.\, \varphi \, i): \]

each injective placement of \(n\) marked indices extends to a full permutation in exactly \((|\alpha |-n)!\) ways, so the right-hand side does not depend on \(e\). (In Lean the coefficient is the \(\mathbb {N}\)-scalar action on the AddCommMonoid \(M\); for the \(\mathbb {Q}\)-valued weights used downstream this is ordinary multiplication.) This is generic finite combinatorics, entirely independent of the ORP construction.

Formerly axiomatized here; now fully proven (PermFiber.lean), so #print axioms sum_perm_comp_embedding \(=\) [propext, Classical.choice, Quot.sound].

Theorem 32Reduction packaging (axiom: meta-fact)

A deterministic polynomial-time many-one reduction that is correct in both directions from an NP-complete source problem, and whose output has polynomially bounded size, witnesses NP-hardness of the target. Axiomatized meta-fact: stands in for the complexity-theoretic framework (poly-time many-one reductions, the NP hierarchy) that Mathlib does not provide. The explicit polynomial size-bound hypothesis is what makes the poly-time character load-bearing in the type.

Definition 3FAST objective and decision problem

For a linear order \(\pi \in S_V\) (an Equiv.Perm of \(V\), with \(\pi (u)\) the position of \(u\)), the feedback-arc-set count is \(\operatorname {fas}(T,\pi ) = |\{ (u,v) \in A : \pi (u) {\gt} \pi (v)\} |\). The FAST decision predicate is \(\mathrm{FAST}(T,k) :\Leftrightarrow \exists \pi ,\ \operatorname {fas}(T,\pi ) \le k\).

Definition 6Posterior cost and the ORP decision

The posterior cost of a candidate order \(\hat\pi \) is \(\mathrm{cost}(\hat\pi ) = Z^{-1}\sum _\sigma w(\sigma )\, d_{\mathrm{KT}}(\hat\pi ,\sigma )\) (ORPInstance.cost). Definition 14\(\alpha _u\) and the reduction \(f(T,k)\)

Set \(L = 2nR\) and \(q = L - n\). The instance has one designated event \(d_u\) per vertex and \(q\) filler events; \(\Delta = 1\) and every observation is \(0\). View the \(L\) slots as \(R\) blocks of width \(2n\); in block \(b\) vertex \(u\) owns the consecutive slot pair \(a^-_{u,b} = (b-1)2n + 2\rho _b(u) - 1\), \(a^+_{u,b} = (b-1)2n + 2\rho _b(u)\). With \(\alpha _u = \tfrac 12 - \tfrac {2\delta _u}{R} + \tfrac {\delta _u}{R^2}\), the designated noise is \(F_{d_u}(-a^-_{u,b}) = (1-\alpha _u)/R\), \(F_{d_u}(-a^+_{u,b}) = \alpha _u/R\) (\(b = 1,\dots ,R\)), and each filler has the uniform law \(F_f(-s) = 1/L\) (\(s = 1,\dots ,L\)). The reduction maps \((T,k)\) to this instance together with threshold \(K = C_0 + 2k/R^2\) (orpThresholdOf; see Lemma 29). The builder is parameterized in \(\rho \) so the heavy Layer-4/5 lemmas can take the gadget properties as hypotheses; the public orpOfTournament fixes \(\rho \) via profile_gadget.choose.

Definition 5Posterior weight, normalizer, and pair marginal

An ORP instance carries \(L\) finite-support rational noise PMFs \(F_i : \mathbb {Z} \to \mathbb {Q}\). With every observation equal to \(0\), the unnormalized posterior weight of \(\sigma \in S_L\) is \(w(\sigma ) = \prod _{i} F_i(-\sigma (i))\) (ORPInstance.weight), normalized by \(Z = \sum _\sigma w(\sigma )\) (ORPInstance.Z). For events \(i,j\) the pair marginal is \(p_{ij} = \mathbb {P}(\sigma (i) {\lt} \sigma (j)) = Z^{-1}\sum _{\sigma : \sigma (i) {\lt} \sigma (j)} w(\sigma )\) (ORPInstance.pairLess).

Definition 10Profile construction

Set \(m = \binom n2\) and \(R = 2m + 8n\). The profile \(\rho : \mathrm{Fin}\, R \to \mathrm{Perm}(\mathrm{Fin}\, n)\) is the concatenation of: (a) for each of the \(m\) arcs \((u,v) \in A\), the two gadget orders \(O_1, O_2\) of Lemma 9 (\(2m\) orders, extracted by choice); (b) \(4n\) copies of the identity and \(4n\) copies of its reverse \(\mathrm{rev}\) (the neutral pairs \((\tau , \tau ^{\operatorname {rev}})\)). Ranks are 0-indexed, \((\rho _b(u) : \mathbb {N}) \in \{ 0,\dots ,n-1\} \). This is hand-authored in Lean as a choice-based construction; the development tracks \(R = 2m + 8n\) directly rather than proving \(m = \binom n2\) (downstream only needs even / \({\ge }\, 8n\) / \({\le }\, 2n^2+8n\)).

Definition 1Tournament and degree difference

A tournament on \(V = \{ 1,\dots ,n\} \) (the Lean index type Fin n, i.e. \(\{ 0,\dots ,n-1\} \)) is a relation \(\mathrm{beats} : V \to V \to \mathrm{Bool}\) that is irreflexive (\(\mathrm{beats}(u,u) = \mathrm{false}\)) and exactly one-directional on each unordered pair (\(u \ne v \Rightarrow \mathrm{beats}(u,v) = \neg \, \mathrm{beats}(v,u)\)). Arc \((u,v) \in A\) means \(\mathrm{beats}(u,v) = \mathrm{true}\). For \(u \in V\) the degree difference is \(\delta _u := \operatorname {indeg}(u) - \operatorname {outdeg}(u)\) (Tournament.degDiff).

Lemma 21Designated–designated posterior margins

Let \(B_u\) be the block of \(X_u\); by independence the \(B_u\) are i.i.d. uniform on \(\{ 1,\dots ,R\} \). Then \(\mathbb {P}(X_u {\lt} X_v) = \frac{R(R-1)/2 + c_{uv}}{R^2}\), which equals \(\tfrac 12 + \tfrac 1{R^2}\) if \((u,v) \in A\) and \(\tfrac 12 - \tfrac 1{R^2}\) if \((v,u) \in A\). So each designated pair’s posterior majority direction is the tournament arc direction, with decision weight \(|2p-1| = 2/R^2\) and constant term \(\min (p,1-p) = \tfrac 12 - \tfrac 1{R^2}\).

Lemma 9Single-arc gadget margins and rank sums

For an arc \((u,v)\) pick any order \(\omega \) of \(V \setminus \{ u,v\} \) and form the two orders \(O_1 = (u,v,\omega )\) and \(O_2 = (\omega ^{\operatorname {rev}}, u, v)\). Then: (i) both rank \(u\) before \(v\); (ii) every ordered pair other than the arc itself (in either orientation) is reversed between \(O_1\) and \(O_2\) (net margin \(0\)); (iii) the two orders give \(u\) the 0-indexed rank sum \(n-2\) (ranks \(0\) and \(n-2\)), \(v\) the rank sum \(n\) (ranks \(1\) and \(n-1\)), and every \(x \notin \{ u,v\} \) the rank sum \(n-1\).

Lemma 18Exact independence via fillers

Conditioned on \(\hat t = 0\), the designated slots are mutually independent, with \(\mathbb {P}(X_u = a^-_{u,b}) = (1-\alpha _u)/R\) and \(\mathbb {P}(X_u = a^+_{u,b}) = \alpha _u/R\), where \(X_u\) is the slot of \(d_u\). The pairwise joint law factors: \(\mathbb {P}(X_u = s \wedge X_v = t) = \mathbb {P}(X_u=s)\, \mathbb {P}(X_v=t)\) for \(u \ne v\).

Lemma 8Majority form of a pairwise contribution

Write the contribution of pair \(\{ i,j\} \) to \(\mathrm{cost}(\hat\pi )\) as \(\ell _{ij}(\hat\pi ) = p_{ij}\, \mathbf{1}[\hat\pi (i){\gt}\hat\pi (j)] + (1-p_{ij})\, \mathbf{1}[\hat\pi (i){\lt}\hat\pi (j)]\). Calling the more likely relative order the posterior majority order (ties broken arbitrarily),

\[ \ell _{ij}(\hat\pi ) = |2p_{ij}-1|\, \mathbf{1}[\hat\pi \text{ disagrees with the majority order on } \{ i,j\} ] + \min (p_{ij}, 1-p_{ij}). \]

In particular, if \(p_{ij} = 1/2\) the contribution is always \(1/2\), independent of \(\hat\pi \).

Lemma 30Polynomial size bounds

\(m \le n^2\), \(R = 2m + 8n \le 2n^2 + 8n\), and \(L = 2nR \le 4n^3 + 16n^2\); in particular \(L \le 16(n+1)^3\), so the reduction’s output is polynomial-sized.
